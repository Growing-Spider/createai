眼底图像AI教学与科研辅助平台 — 知识库说明
==============================================

本目录用于存放RAG（检索增强生成）知识库文档。

支持格式
--------
- PDF文件（.pdf）：教材、期刊论文、指南
- 文本文件（.txt）：知识摘要、笔记
- Word文档（.docx）：报告、论文草稿

建议内容
--------
1. 眼科学教材章节（视网膜疾病、血管结构）
2. 医学图像处理相关教材
3. 近5年高质量SCI论文摘要
4. 糖尿病视网膜病变诊断指南
5. 深度学习医学影像分析综述

内置知识
--------
系统已内置以下基础知识（无需上传文件即可使用）：
- 眼底图像基础知识
- 糖尿病视网膜病变分期
- U-Net模型原理
- 血管拓扑特征说明
- RAG技术原理
- 青光眼诊断要点
- 医学图像评估指标

扩展知识库
----------
1. 在Streamlit侧边栏「知识库管理」上传文档
2. 点击「重建知识库」完成向量化索引
3. 文档将自动分块并存储至ChromaDB

推荐数据库/来源
--------------
- PubMed (https://pubmed.ncbi.nlm.nih.gov)
- IEEE Xplore (https://ieeexplore.ieee.org)
- Nature Medicine
- Ophthalmology (American Academy of Ophthalmology)
- 中华眼科杂志
